## What This Product Does

`IELTS Speaking AI` helps learners practice IELTS speaking with both full mock exams and shorter Free Practice sessions.

## Before You Start

- use a browser environment with microphone access
- make sure audio recording permissions are allowed
- depending on the deployment, you may need to log in before starting a session

## Main Entry Paths

- **Full mock exam**: practice Part 1, Part 2, and Part 3 as one connected flow
- **Free Practice**: pick a topic from the library or write your own prompt and start speaking directly

## What Happens After Practice

After a session, the platform can show transcripts, scoring feedback, and related history so users can review progress over time.

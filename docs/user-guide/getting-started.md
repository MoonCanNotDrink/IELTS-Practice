## 这个产品做什么

`IELTS Speaking AI` 帮助学习者进行 IELTS 口语练习，既支持完整模考，也支持更轻量的自由练习。

## 开始前需要准备什么

- 使用支持麦克风权限的浏览器环境
- 确保已经允许音频录制权限
- 视部署方式而定，开始练习前可能需要先登录

## 主要入口路径

- **完整模考**：将 Part 1、Part 2、Part 3 作为一套连续流程来练习
- **自由练习**：从题库中选择题目或自己输入提示词，然后直接开始回答

## 练习后会发生什么

一轮练习结束后，平台可以展示转写、评分反馈和历史记录，帮助用户持续复盘。

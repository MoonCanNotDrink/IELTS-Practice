## Is this only for full mock exams?

No. The platform supports both full mock exams and a separate Free Practice mode.

## Can I use my own prompt?

Yes. Free Practice supports custom prompts in addition to topic-library selection.

## Does the platform keep history?

Yes. The project includes history and review capabilities so users can revisit previous sessions.

## Is this only a prototype?

No longer. The current system has grown beyond a narrow demo and now connects practice flow, feedback, and review in a more complete training loop.

## What It Covers

Free Practice is the faster, lower-friction path for users who want to practice speaking without going through the full exam flow.

## How It Works

Users can:

- choose a topic from the topic library
- search through grouped topics
- write a custom prompt instead
- adjust speaking duration before answering

## Why It Matters

This mode is useful for repeated, targeted speaking practice and for shorter study sessions.

# 文档风格指南

## 目标

本指南用于统一项目文档的导航样式、标题层级和术语表达。

## 命名规则

- 中文文档使用无后缀文件名，例如 `README.md`
- 英文文档使用 `.en.md` 后缀，例如 `README.en.md`

## 导航规则

- 文档顶部优先提供 `导航`
- 默认优先链接到同语言页面
- 如果存在另一语言版本，应显式提供跨语言入口
- 用户指南类文档底部统一使用 `继续阅读`

## 标题层级

- 每个文档只保留一个 `#` 主标题
- 主要章节使用 `##`
- 次级章节使用 `###`
- 避免跳级使用标题

## 中文术语

优先使用以下术语：

- `完整模考`
- `自由练习`
- `题库`
- `文档目录`
- `项目简介`
- `对外文档`
- `用户指南`
- `快速开始`
- `常见问题`
- `继续阅读`

## English Terms

中文文档中引用英文页面时，优先对应以下英文名称：

- `Home`
- `Documentation Index`
- `Project Overview`
- `External Documentation`
- `User Guide`
- `Getting Started`
- `Mock Exam Guide`
- `Free Practice Guide`
- `FAQ`
- `Continue Reading`
